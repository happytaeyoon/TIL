```<!DOCTYPE html>```
문서의 HTML 버전 지정<br>

```<html>```
HTML 문서가 어디에서 시작하고, 어디에서 끝나는지 알려줌<br>

```<head>```
문서의 정보를 나타내는 범위<br>

```<body>```
문서의 구조를 나타내는 범위<br>

```<title>```
제목을 정의<br>

```<link rel="" href="">```
외부 문서를 가져와 연결 (대부분 CSS파일)<br>

```rel``` 가져올 문서와 관계
href 가져올 문서의 경로<br>

```<style>``` 스타일을 문서안에서 직접적으로 작성할때<br>

```<script src="">``` 자바스크립트 파일을 가져올때<br>

```<meta>``` 제작자,내용,키워드 같은 여러 정보를 검색엔진이나 브라우저에게 제공<br>

상대경로 vs 절대경로
./ 주변 ../ 상위폴더<br>

```<a>``` 주로 링크 달때 사용<br>

Emmet, ```div.hi=<div class="hi">``` w:200 = width:200px; bc:orange=background-color:orange, ul>li*4<br>

```<태그>내용</태그>``` : 요소(Element), 이때 내용을  자식요소, 태그를    부모요소라 한다<br>

닫히는 태그 가 없는 태그: Empty태그
<br>

인라인(inline)요소: 글자를 먼들기 위한 요소, 가소 세로가 콘텐츠 크기만큼 줄어들음, 가로세로 사이즈 지정할 수 없다,위 아래 여백 적용 x, 블록요소를 같이 쓸수 x<br>

```<span>```: 본질적으로 아무것도 나타내지 않음, 콘텐츠 영역을 설정하는 요소, 요소가 수평으로 쌓임,span두개를 이어쓰면 띄어쓰기x<br>

```<img>```: 사진 넣는 태그, src 경로, alt 이름<br>

```<a>```:다른/같은 페이지로 이동하는 하이퍼링크를 지정하는 요소(Anchor),href:링크 target:표시위치,_blank:새탭<br>

```<br>```:줄바꿈요소<br>

```<label>```:라벨 가능 요소(input)의 제목<br>

### 블록(Block)요소: 상자(레이아웃)을 만들기 위한 요소<br>

```<div>```:본질적으로 아무것도 나타내지 않음, 콘텐츠 영역을 설정하는 요소, 요소가 수직으로 쌓임,부모요소의 크기만큼 자동으로 늘어남(가로)<br>

```<h1>```: 1~6 Heading 제목을 의미하는 요소<br>

```<p>```: 문장을 의미하는 요소<br>

```<ul>```:순서가 필요없는 목록의 집합(unordered list),```<li>```:목록 내 각 항목(list item)<br>

```<tabel>```:표요소, 행과 열의 집합,행tr,열td(table data)<br>


### inline-block:
### 수평으로 쓸수 있으며 가로세로 여백지정가능<br>

```<input>```: 사용자가 데이터를 입력하는 태그 type을 지정해야함,label로 이름지정가능,text,checkbox(checked:미리 체크),radio:택1,name:그룹으로 묶기 value:미리 입력되어있을 값 placeholeder:사용자가 입력할 값의 힌트,disabled:비활성화
<br>

래핑(wrapping):요소(텍스트)를 어떤 요소로 묶어주는 행위<br>

주석 ctrl+/<br>


### 전역속성: 
title(요소의 정보나 설명을 지정), style(요소에 적용할 스타일을 지정),class(요소를 지칭하는 중복이 가능한 이름),id(요소를 지칭하는 고유한 이름),data((요소에 데이터를 지정) ex:data-fruit-name="banana")
